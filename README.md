## Цель работы
Научиться поднимать веб-сервер Nginx в контейнере Docker и отдавать простую HTML-страничку.

## Ход работы

### Этап 1: Подготовка окружения
- Установлен Docker Desktop
- Проверка версий:
docker --version
docker-compose --version
- **Скриншот 1:** Версии Docker

### Этап 2: Первый запуск Nginx
- Создана папка `nginx-lab`
- Создан `docker-compose.yml`
- Запуск: `docker-compose up -d`
- Проверка: http://localhost:8080
- **Скриншот 2:** Стандартная страница Nginx

### Этап 3: Добавление своего сайта
- Создана папка `code` с файлом `index.html`
- Обновлён `docker-compose.yml` с `volumes`
- Перезапуск: `docker-compose up -d --build`
- **Скриншот 3:** Моя страница "Hello from Docker & Nginx"

### Этап 4: Эксперименты
1. Изменение текста в index.html
 - **Скриншот 4:** Изменённая страница
2. Создание about.html
 - **Скриншот 5:** Страница about.html
3. Смена порта 8080 → 3000
 - **Скриншот 6:** Работа на порту 3000

## Вывод
Веб-сервер Nginx успешно запущен в Docker. Настроена отдача пользовательских HTML-страниц. Контейнер работает стабильно, изменения в файлах применяются без перезапуска контейнера.

## Исходный код

### Файл `docker-compose.yml`
```yaml
version: "3.9"
services:
web:
  image: nginx:1.27-alpine
  ports:
    - "8080:80"
  volumes:
    - ./code:/usr/share/nginx/html
